# Data Analysis Using Python and pandas

## Overview
This project analyzes expense data to extract meaningful insights using Python and Pandas. 
It focuses on category-wise, city-wise, and time-based spending analysis.

## Features
- Category-wise expense aggregation
- Monthly spending trend analysis
- City-wise spending distribution
- Payment method usage analysis
- Data visualization using Matplotlib(pyplot)

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the analysis:
   python analysis.py
